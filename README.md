**Author:** Pranesh A | Guru Nanak College (Autonomous)
**Supervisor:** Dr. E. Gayathiri | March 2026

## Project Summary
10-step integrated pipeline: GSE30122 transcriptomics → ADMET → 
Molecular Docking → 1 ns GROMACS MD simulation.
Targets: Keap1/Nrf2 · NF-κB · TGF-β1

## Tools Used
GROMACS 2023 · AutoDock4 · PyMOL · RStudio · SwissADME · ProTox-II

## Key Results
- 1,247 DEGs identified (limma, adj.p<0.05)
- Best docking: Quercetin–TGF-β1 → −7.5 kcal/mol
- RMSD equilibration: 0.15 nm (1 ns MD, Myricetin–3BRT)

## Data Source
GSE30122 — NCBI GEO (publicly available)
